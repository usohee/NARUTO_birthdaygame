# 🥷 나루토 생일 축하 미니게임 (Naruto Birthday Game)

남자친구의 생일을 축하하기 위해 제작한 **나루토 컨셉의 웹 미니게임**입니다. 
Vanilla JS를 활용하여 영화 같은 화면 전환과 픽셀 아트 감성을 구현했습니다.

## 🛠 사용 기술 (Tech Stack)
- **Language:** HTML5, CSS3, Vanilla JavaScript
- **Font:** Mulmaru (웹 폰트 적용)
- **Design:** Figma, Pixel Art Assets

## 🎮 주요 기능 (Features)
- **Story Scene:** 닌자 등록증 발급 및 독백 연출 (Fade In/Out 효과)
- **Training Scene:** 카카시 선생님과의 화둔 훈련 대화창 구현
- **Mini Game:** (여기에 미니게임의 핵심 룰을 적으세요!)

## 📂 프로젝트 구조
- `index.html`: 인트로 화면
- `ID_card.html`: 닌자 등록증 및 독백 화면
- `minigame.html`: 미니게임 준비 및 대화 화면
- `assets/`: 이미지, 폰트, 사운드 리소스
- `ground/`: 배경 이미지
